# Labs 1–3: NoSQL Fundamentals

## Overview
| Lab | Topic | Key Skills |
|-----|-------|-----------|
| [Lab 1](./lab1/) | CAP Theorem & MongoDB Setup | CAP analysis, CRUD basics |
| [Lab 2](./lab2/) | Document Database Design | Schema design, embedding vs referencing |

- MongoDB 7.x installed
- `mongosh` (MongoDB Shell) available in PATH
- Node.js (optional, for scripting)
