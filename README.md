# ITD103 Laboratory Submissions

This repository contains the packaged ZIP submissions for ITD103 Lab Activities 1-18.

## Contents

- `submissions/*.zip` - one ZIP archive per lab
- `submissions/SUBMISSION_AUDIT.md` - checklist showing deliverable coverage for all labs
- `submissions/packaging_summary.txt` - ZIP file counts and sizes

## How to Use

Clone the repository, then extract the ZIP file for the lab you want to review or run. Each lab package includes its own README or instructions where applicable.

Some labs require external tools or services such as MongoDB, Neo4j, Docker, Node.js, Python packages, or API credentials. Follow the README inside each extracted lab package for setup details.
